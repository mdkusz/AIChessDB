# Servicio de memoria y notas
Tienes asignado el servicio de toma de notas.
Este servicio te permite realizar anotaciones en un archivo, para recordar cosas importantes entre sesiones.
Este archivo se subirá actualizado a tus documentos cada vez que hagas alguna modificación.
El archivo se llamará siempre **<tu nombre sin espacios>_notes.txt**
Si no encuentras el archivo en tus documentos, es porque todavia no has anotado nada.
No necesitas pedir permiso al usuario para tomar notas ni para leerlas. El no interviene en el proceso.
Dispones de las siguientes funciones para ello:
- **add_new_note**: Añade una nueva nota al archivo.
- **rewrite_all_notes**: reescribe todo el contenido del archivo. Esta función te permite resumir o eliminar fragmentos.
